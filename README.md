## Description

**Chic Spaces** is a modern, responsive, and full-featured **furniture e-commerce website** built using **ASP.NET Core MVC**.  
This is my **3rd full-stack project** and my **1st ASP.NET website**, showcasing my skills in backend development, database management, and frontend design.  

The website provides a seamless user experience with **dynamic product listings, trending product carousels, categories, hero videos, and newsletter subscription**.  
Users can **register, login, manage their profile, add products to the cart, and browse products by category**.  

Admins have a dedicated **dashboard** to manage **products, categories, sliders, pages, and orders**, making the site fully dynamic and easily maintainable.  

The design emphasizes a **warm and modern aesthetic**, with hover effects, smooth carousels, and responsive layouts for **desktop and mobile devices**.  

---

## Key Features

- Multi-step **user registration and profile management**  
- **Role-based authentication** (User/Admin)  
- **Trending products carousel** & category-based navigation  
- **Hero/feature video** for visual engagement  
- **Newsletter subscription** integration  
- **Responsive and interactive UI** with Bootstrap and custom CSS  
- **Admin dashboard** for full content management  
- Add-to-cart functionality and order management for users  

---

## Technologies Used

- **Backend:** ASP.NET Core MVC  
- **Database:** Entity Framework Core + SQL Server  
- **Frontend:** HTML5, CSS3, Bootstrap 5, JavaScript, jQuery  
- **Authentication:** Cookie-based authentication with roles
